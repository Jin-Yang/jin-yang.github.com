## 环境准备

安装所需要的包，并启动。

{% highlight text %}
$ npm install
{% endhighlight %}

根据 `package.json` 中的配置安装所需要的包，此时会在本地目录下安装 `webpack-dev-server` 包，本地可以通过如下方式启动。

{% highlight text %}
$ ./node_modules/.bin/webpack-dev-server
{% endhighlight %}
